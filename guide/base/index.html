快速上手 ​

提示

turing-ui-vue3 是基于 vue3 + ts + Element-plus 二次封装的基础组件

安装 ​

bash
pnpm add turing-ui-vue3 -S
&
npm install turing-ui-vue3 -S

全局使用 ​

前提条件：使用项目必须全局注册 Element-plus 组件库 ​

element-icon也需要全局引入，否则部分使用到el-icon的图标不能正常显示出来

js
// main.ts
import { createApp } from "vue"
import App from "./App.vue"
/* 引入element-plus */
import ElementPlus from 'element-plus';
import 'element-plus/dist/index.css';
import * as ElementPlusIconsVue from '@element-plus/icons-vue'
import locale from "element-plus/es/locale/lang/zh-cn"
/* 引入组件库（基于element-plus二次开发） */
import TuringUI from "turing-ui-vue3"
import "turing-ui-vue3/lib/style.css"

const app = createApp(App)
// 注册所有图标
for (const [key, component] of Object.entries(ElementPlusIconsVue)) {
  app.component(key, component)
}
// 注册ElementPlus
app.use(ElementPlus, {
  locale // 语言设置
  // size: Cookies.get('size') || 'medium' // 尺寸设置
})
app.use(TuringUI, { locale: 'zh-cn' }) // locale默认值是zh-cn,可以不传
app.mount("#app")

按需引入 ​

js
// 在main.js中按需引入
import 'turing-ui-vue3/lib/style.css'
// 单个.vue文件引入
<script setup lang="ts">
  import {TuButton, TuStatus} from "turing-ui-vue3"
</script>
// 在main.js中按需引入
import 'turing-ui-vue3/lib/style.css'
// 单个.vue文件引入
<script setup lang="ts">
  import {TuButton, TuStatus} from "turing-ui-vue3"
</script>

Git 提交规范（PR 提交规范） ​

  • ci: ci 配置文件和脚本的变动;
  • chore: 构建系统或辅助工具的变动;
  • fix: 代码 BUG 修复;
  • feat: 新功能;
  • perf: 性能优化和提升;
  • refactor: 仅仅是代码变动，既不是修复 BUG 也不是引入新功能;
  • style: 代码格式调整，可能是空格、分号、缩进等等;
  • docs: 文档变动;
  • test: 补充缺失的测试用例或者修正现有的测试用例;
  • revert: 回滚操作;
