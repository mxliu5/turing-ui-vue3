主题定制 ​

通过css变量覆盖element-plus的默认变量

css
:root {
  --el-color-primary: #177BF8;
}

出于性能原因，更加推荐你在类名下添加自定义 css 变量，而不是在全局的 :root 下。 如：

css
.turing-ui-frame {
  --el-color-primary: #177BF8;
  ...
}

在你的项目中自定义主题步骤 ​

  1. 新建theme.scss文件
css
/* 全局css变量，实现一键换肤[这里的样式变量会覆盖element-plus对应的样式变量] */

  1. 新建vars.scss
css
/* 全局scss变量，方便在scss中直接使用scss变量定义样式 */

// 主题色
$primary-color: var(--el-color-primary);
$warning-color: var(--el-color-warning);
$success-color: var(--el-color-success);
$danger-color: var(--el-color-danger);
$info-color: var(--el-color-info);

// 字体色
$title-color: var(--el-text-color-primary);            // 标题色
$text-color: var(--el-text-color-regular);             // 文本色
$text-color-secondary: var(--el-text-color-secondary); // 次级文本色
$text-color-third: #81878C;                          // 三级文本色
$disabled-color: var(--el-text-color-disabled);        // 禁用色
$placeholder-color: var(--el-text-color-placeholder);  // placehodler色

// 边框
$border-color: var(--el-border-color);                          // 边框
$border-color-split: var(--el-border-color-light);              // 分割线
$border-color-split-lighter: var(--el-border-color-lighter);    // 分割线[更浅]
$border-radius: var(--el-border-radius-base);                   // 圆角
  1. 如果你正在使用vite
ts
import { defineConfig } from 'vite'
export default defineConfig({
  css: {
    // 添加css预处理器配置
    preprocessorOptions: {
      scss: {
        // additionalData的内容会在每个scss文件的开头自动注入，这样就可以全局使用scss了
        additionalData: '@use "@/styles/vars.scss;'
      }
    }
  }
})
  1. 如果您正在使用 webpack
ts
import { defineConfig } from 'vite'
export default defineConfig({
  css: {
    loaderOptions: {
      scss: {
        additionalData: '@use "@/styles/vars.scss;'
      }
    }
  }
})
  1. 在scss文件中或者<style lang="scss" scoped></style>中使用scss变量
css
.link-text {
  color: $primary-color;
}
