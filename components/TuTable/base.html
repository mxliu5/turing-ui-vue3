Table 表格 ​

基础用法（在el-table的基础上内置了一些常用功能） ​

继承el-table所有属性、插槽、方法以及事件。所有写法一致 参考：el-table使用说明文档

  1. 通过设置custom: 'link'来设置可点击的超链接文本
  2. 通过设置custom: 'status'来设置状态项
  3. 通过设置custom: 'tas-status'来设置带有背景色的状态项，可以通过tagAttrs来设置tag的属性
  4. 通过设置custom: 'switch'来设置开关项
  5. 通过设置widthCopy: true来设置可复制的项
  6. 如果状态项需要展示标签提示，则需要设置customRender: {tips: 'rejectReason'}

序号 ​

通过with-order属性展示序号
如果表格有分页需求，需要配置order-option属性，将当前页数pageNum和每页查询的条数pageSize传过去，否则翻页时不能计算出叠加的序号。如果表格没有分页需求，无需设置

多选框 ​

通过with-selection属性展示多选框
可以通过selection-option来设置多选框列的相关属性，如width、selectable、reserve-selection等

  1. selectable用来设置当前行是否支持勾选
  2. reserve-selection用来设置翻页时，不清空上一页数据。必须指定row-key的值
  3. toggleRowSelection用来选中行
  4. clearSelection用来清空选中项 更多使用请参考 el-table使用说明文档

固定表头和列 ​

  1. 通过设置表格的height属性来固定表头，一般情况下这个值可根据页面高度自适应计算
  2. 通过设置列的fixed值固定列，left为固定在左侧，right或true为固定在右侧。
  3. 在设置表格项宽度width时，需要至少保留一个width不是固定的，可以用minWidth代替，否则可能出现大屏表格宽度未到100%的情况
  4. 序号和selection列默认是固定在左侧的

名称可编辑（鼠标移入显示编辑图标） ​

  1. 通过设置表格custom: 'editInput'来设置可编辑项，鼠标移入显示编辑按钮，点击编辑按钮出现文本框
  2. 通过设置customRender: {attrs: maxlength: 20}来设置文本框的属性，继承el-input所有属性

操作项禁用或隐藏、更多操作项设置 ​

  1. 通过设置disabled来控制表格项是否禁用，可通过disabledTips设置禁用的原因提示
  2. 通过设置exist来控制表格项是否显示
  3. 通过设置collapsed: true将操作按钮展示在更多中

行拖拽 ​

  1. 通过设置draggable来控制表格行是否支持拖拽
  2. 通过draggableOption配置拖拽图标单元格的属性，如果没配置不显示拖拽图标，默认鼠标选中当前行进行拖拽
  3. 可以通过dragIcon插槽来实现拖拽图标的自定义

提示信息：

必须要设置row-key，否则拖拽会错乱

表格数据为空 ​

插槽 ​

  1. 通过<template #empty></template>设置表格空值自定义配置
  2. 通过<template #列的字段名prop-header></template>定义当前列的头部插槽内容
  3. 通过<template #列的字段名prop-filer-icon></template>定义当前列的筛选项的icon图标
  4. 通过<template #列的字段名prop="scope"></template>定义当前列单元格的插槽内容，需要将custom设置为'slot'
  5. 同第4点一样，给操作列设置custom: slot来自定义操作栏

提示信息：

插槽返回的scope值为{row, $index}，row为当前行的值，$index为当前行的下标

customRender配置说明 ​

下面分别对不同的custom类型的customRender配置进行列举

custom: 'link' ​
属性名说明
attrs文本按钮的属性，继承tu-text-button所有属性
click文本点击事件，返回当前行的数据
disabled可根据当前行某个字段的值取区分是否禁用
javascript
{ 
  prop: "name", 
  label: '应用名称',
  custom: 'link',
  customRender: {
    click: (record: any) => {
      console.log('跳转到详情页', record)
    },
    // tu-text-button的属性
    attrs: {},
    // 禁用
    disabled: (record: any) => record.num === 0
  } 
}

custom: 'status' ​
属性名说明类型
options状态项集合，格式参考demoobject
tips状态的提示信息function
custom: 'tag-status' ​
属性名说明类型
options状态项集合，格式参考demoobject
tips状态的提示信息function
tagAttrs标签的属性，继承el-tag是所有属性，可以通过修改属性来定义tag样式object
javascript
{ 
  prop: "status", 
  label: "tag状态", 
  custom: 'tag-status',
  customRender: {
    options: {
      1: { type: "success", name: "已发布" },
      2: { type: "info", name: "未发布" },
      3: { type: "warning", name: "待审核" },
      4: { type: "primary", name: "审核中" },
      5: { type: "danger", name: "已驳回" },
    },
    tips: (row: any) => row.rejectReason,
    // 通过tagAttrs设置tag的属性
    tagAttrs: {
      size: 'small',
      round: true
    }
  }
}

custom: 'switch' ​
属性名说明
attrs开关的属性，继承el-button所有属性
beforeChange开关切换之前的事件，一般在这里执行开关操作后的逻辑
disabled可根据当前行某个字段的值取区分是否禁用
javascript
{ 
  prop: "enable", 
  label: '启用状态', 
  width: 100, 
  custom: 'switch',
  customRender: {
    beforeChange: (record: any) => {
      // resolve(true) 改变状态  reject()状态不变 用于禁用确认框提示，如果确认后并且接口调用成功resolve(true),如果击取消获取接口调用失败reject()
      return new Promise((resolve: any, reject: any) => {
        resolve(true)
      })
    },
    // 开关的属性
    attrs: {},
    // 禁用
    disabled: (record: any) => record.status === 1
  }
}

custom: 'editInput' ​
属性名说明
attrsinput的属性，继承el-input所有属性
blur文本框失焦事件，在这里执行名称修改的逻辑
enter文本框enter事件，在这里执行名称修改的逻辑
disabled可根据当前行某个字段的值取区分是支持修改，如果不支持，则鼠标移入没有编辑按钮
javascript
{ 
  prop: "name", 
  label: "应用名称", 
  width: 180,
  showOverflowTooltip: false,
  custom: 'editInput', // 名称支持编辑
  customRender: {
    blur: ( row: any) => {
      // 失焦时进行接口调用，修改名称
      console.log(row)
      // 名称修改成功后刷新列表
    },
    enter: (row: any) => {
      // 失焦时进行接口调用，修改名称
      console.log(row)
      // 名称修改成功后刷新列表
    },
    // el-input的属性
    attrs: {
      maxlength: 20
    },
    // 当前名称是否支持编辑，输入鼠标，不显示编辑按钮
    disabled: (record: any) => record.name === '默认名称'
  }
}
