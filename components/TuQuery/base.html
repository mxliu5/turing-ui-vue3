Query 查询组件 ​

基础用法 ​

  1. 支持element-plus所有的表单控件，通过type:'控件名'来定义查询项类型

:::demo TuQuery/base :::

支持折叠 ​

  1. 通过support-fold属性设置“支持折叠”功能，默认至为false

:::demo TuQuery/fold :::

即时搜索 ​

不需要“查询”、“重置”按钮，表单控件操作完成后即时搜索
不设置searchBtn属性，或者searchBtn={show: false}即可隐藏搜索按钮，并且当表单控制操作完成后直接执行查询功能 :::demo TuQuery/noButton :::

文本框后置搜索按钮 ​

通过给文本框查询项设置showAppend: true来实现后置搜索按钮的文本框，enter以及点击搜索按钮均可以执行查询功能 :::demo TuQuery/showAppend :::

展示刷新按钮 ​

通过refresh-btn={show: true}展示刷新按钮 :::demo TuQuery/refresh :::

查询/重置按钮设置 ​

  1. 通过search-btn={right: true}来设置按钮是否固定在右侧
  2. 通过search-btn={searchName: '搜索', resetName: '清除'}来自定义查询和重置按钮的文本 :::demo TuQuery/searchBtn :::

复杂控件 ​

  1. 通过type: 'quickDate'设置快速日期控件
  2. 通过type: 'slot', slotName: '插槽名称'来自定义tu-query没有内置的控件

INFO

使用较频繁的控件直接在tu-query中内置
使用不频繁的控件通过插槽自定义

:::demo TuQuery/custom :::

label自定义 ​

  1. 通过设置labelColon: false隐藏标签的冒号，默认值为true
  2. 通过设置查询项属性labelRender来自定义label（render 函数 jsx 方式编写） :::demo TuQuery/label :::

Query属性 ​

属性名说明类型默认值
queryItems查询项的集合object见下方queryItems属性说明
labelColon是否显示label标签的冒号“：”，booleantrue
searchBtn查询/重置按钮相关配置object见下方searchBtn属性说明
refreshBtn刷新按钮相关配置object见下方refreshBtn属性说明
supportFold查询项过多是否支持“展开收起”功能booleanfalse
foldBtn展开/收起按钮相关配置object见下方foldBtn属性说明

queryItems属性说明 ​

属性名说明类型默认值
modelValue查询项绑定的值--
type查询项类型，支持element-plus所有的表单控件，另外支持自定义的quickDate和slot类型，通过设置type为'slot'来实现自定义插槽查询项string-
childCompName子组件名称（例如单选按钮组type: 'el-radio-group', childCompName: 'el-radio'）string-
slotName插槽名称，需要和type:'slot'一起使用string-
label查询项名称，不赋值则不显示label名称string-
labelRender自定义 label（render 函数 jsx 方式编写）string-
width查询项的宽度stringel-select和el-cascader默认宽度为'160px'，其他的控件如果未设置则自适应
hidden是否隐藏当前查询项booleanfalse(可根据权限设置隐藏逻辑)
options当表单项为下拉列表、级联选择框、多选按钮组、单选按钮组时，options为选项值array[{value: '', lable: ''}]
showAppend文本框是否为后置搜索按钮的风格booleanfalse
withTime当type为quickDate时，设置日期是否支持选择时间段booleanfalse
quick当type为quickDate时，需要设置快速按钮的相关属性object见下方quick属性说明
attrs扩展属性继承当前表单控件所有支持的属性-
events扩展事件继承当前表单控件所有支持的事件-

searchBtn属性说明 ​

属性名说明类型默认值
show是否展示booleanfalse
right是否固定在右侧booleanfalse
searchName查询按钮文本内容string"搜索"
resetName重置按钮文本内容string"重置"
searchAttrs查询按钮的属性，继承el-button所有属性oject-
resetAttrs重置按钮的属性，继承el-button所有属性oject-

refreshBtn属性说明 ​

属性名说明类型默认值
show是否展示booleanfalse

foldBtn属性说明 ​

属性名说明类型默认值
expandName展开按钮名称string"展开"
foldName收起按钮名称string"收起"

quick属性说明 (同DateQuick组件的quick属性一致) ​

属性名说明默认值
modelValue单选按钮组绑定值-
options单选按钮集合value/label形式的数组
attrs单选按钮组的属性集合继承el-radio-group所有的属性

插槽 ​

通过查询项的slotName属性进行插槽内容定义
