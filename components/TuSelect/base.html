禁用 ​

:::demo TuSelect/disabled :::

多选 ​

通过multiple属性设置tu-select支持多选
默认情况下选中值会以 Tag 组件的形式展现， 你也可以设置 collapse-tags 属性将它们合并为一段文字。 您可以使用 collapse-tags-tooltip 属性来启用鼠标悬停折叠文字以显示具体所选值的行为。

:::demo TuSelect/multiple :::

插槽使用 ​

  1. default插槽 自定义选项模版内容
  2. header插槽自定义 下拉列表顶部的内容
  3. footer插槽 自定义 下拉列表顶底部的内容
  4. prefix插槽 自定义 select 组件头部内容
  5. empty插槽 自定义 无选项时的列表展示内容
  6. tag插槽自 定义 select组件标签内容
  7. loading插槽 自定义 select组件loading的内容
  8. label插槽 自定义 select组件loading的标签内容

Select 属性 ​

继承el-select所有的属性，这里只列举常用属性以及新增属性

属性名说明类型默认值
v-model选中项绑定值string / number / boolean / object / array-
clearable是否可以清空选项booleantrue
filterable是否可筛选booleantrue
options选项的数据源，value和label可以通过defaultProps自定义array格式为[{value:'', label: ''}, ...]-
defaultPropsoptions的配置选项,具体见下表defaultProps-
valueInLabelchange时是否将绑定值的其他属性带上(有时候需要获取value以外的属性，如名称)booleanfalse
useVirtual是否开启虚拟列表,参考：Virtualized Select 虚拟化选择器booleanfalse

defaultProps 配置选项 ​

属性名说明类型默认值
value指定选项的值为选项对象的某个属性值stringvalue
label指定选项显示的名称为选项对象的某个属性值stringlabel
disabled指定选项的禁用为选项对象的某个属性值stringdisabled

Select Expose ​

继承el-select所有的Expose

插槽名说明
focus使选择器的输入框获取焦点
blur使选择器的输入框失去焦点，并隐藏下拉框
selectedLabel获取当前选中的标签
$refsel-select元素，可通过$ref获取el-select的所有属性
