防抖功能 ​

自定义按钮类型 ​

除了按钮自带的类型success、primary、warning、danger、info以外 还内置了几个常用的业务按钮类型：新增add、导入import、导出export、删除delete

:::demo TuButton/custom-type :::

WARNING

前提是必须在项目中全局引入了element-icon

javascript
import * as ElementPlusIconsVue from '@element-plus/icons-vue'
// 注册所有elemement-icon
for (const [key, component] of Object.entries(ElementPlusIconsVue)) {
  app.component(key, component)
}

TooltipButton ​

不可点击的按钮，鼠标移入按钮展示tooltip提示信息，鼠标点击按钮弹出warning提示
通过tooltipButton属性来设置是否为TooltipButton按钮 :::demo TuButton/tooltipButton :::

Button属性 ​

继承el-button所有的属性，这里只列举常用属性以及新增属性

属性名说明类型默认值
type按钮类型enum"default"
isDebounce是否开启防抖booleantrue
time防抖时间（一定时间内按钮仅触发一次，防止多次触发）number0(单位ms)
tooltipButton是否为tooltipButton类型的按钮object-

tooltipButton属性 ​

属性名说明类型默认值
clickable是否可直接点击booleantrue
handle按钮操作名称string'操作'
tips自定义提示内容string' '
tooltiptooltip的相关属性object参考el-tooltip文档

提示

默认提示信息为“请选择要{handle}的数据”
若想要自定义提示信息，通过传参tips来自定义

Button Expose ​

继承el-button所有的Expose

插槽名说明
ref按钮 html 元素
size按钮尺寸
type按钮类型
disabled按钮已禁用
shouldAddSpace是否在两个字符之间插入空格
$refsel-button元素，可通过$ref获取el-button的所有属性

特别说明：

另外tu-button解决了el-button点击完按钮后，背景颜色未恢复，仍然处于聚焦的状态。tu-button点击完按钮取消聚焦状态，从而背景颜色恢复正常状态
