QuickDate 快速选择日期范围 ​

基础用法 ​

  1. 内置快速选择按钮分别为“近7天”、“近30天”、“近90天”、“近1年”
  2. 通过quick可以自定义快速选择按钮
  3. 通过with-time属性设置是否支持选择时间，默认为false :::demo TuQuickDate/base :::

WARNING

自定义quick时，quick必须为响应式的变量，否则会导致点击按钮没反应 通过reactive或ref定义都可以

自定义日期格式 ​

通过属性formatk可以自定义日期格式，默认为'YYYY-MM-DD'
如果传参with-time为true，则默认值为'YYYY-MM-DD HH:mm:ss' :::demo TuQuickDate/format :::

尺寸大小 ​

通过属性size设置组件大小 :::demo TuQuickDate/size :::

禁用 ​

通过属性disabled来禁用组件 :::demo TuQuickDate/disabled :::

插槽 ​

继承el-date-picker所有的插槽

  1. 通过range-separator自定义范围分割符内容
  2. 通过prev-month自定义上个月的图标
  3. 通过next-month自定义下个月的图标
  4. 通过prev-year自定义上一年的图标
  5. 通过prev-year自定义下一年的图标

:::demo TuQuickDate/slot :::

属性 ​

属性名说明类型默认值
modelValue绑定值array-
disabled禁用，单选按钮组和日期选择器同时生效booleanfalse
size尺寸，单选按钮组和日期选择器同时生效booleanfalse
width日期选择器的宽度width380px
withTime是否支持选择时间booleanfalse
format显示在输入框中的格式string'YYYY-MM-DD'
valueFormat选中值后获得的值的格式string'YYYY-MM-DD'
disabledDate禁用日期Function-
quick单选按钮组自定义配置object见下方quick属性说明

TIP

默认禁用快速选择按钮的最大日期范围，即最大值按钮为“近一年”，那么只能选择近一年的数据。也可以通过disabledDate进行自定义禁用范围

quick属性说明 ​

属性名说明类型默认值
modelValue单选按钮组绑定的值number7
options单选按钮组的数据array默认值见下方
attrs单选按钮组属性，继承el-radio-group所有属性object-

options默认值为：

ts
[ 
  { value:7, label: $t('time.nearDay', { n: 7 }) },
  { value:30, label: $t('time.nearDay', { n: 30 }) },
  { value:90, label: $t('time.nearDay', { n: 90 }) },
  { value:timeC.getDaysInLastYear(), label: $t('time.nearYear', { n: 1 }) }
]
